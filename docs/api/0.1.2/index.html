Server API

The server API provides functionality for running LwM2M servers, managing client connections, and handling resource operations.

bootstrap

LwM2M Bootstrap Server for device provisioning and initial configuration

startBootstrapServer(bootstrapDeviceCall = null, port = 5783)

Starts the LwM2M Bootstrap Server
Parameters:
bootstrapDeviceCall (function|null) - Optional callback for custom bootstrap handling
port (number) - Port to listen on (default: 5783)
Returns:
object - CoAP server instance

handleBootstrap

Bootstrap request handling and device configuration management

handleBootstrapRequest(req, res, bootstrapDeviceCall)

Processes bootstrap requests from LwM2M clients
Parameters:
req (object) - CoAP request object
res (object) - CoAP response object
bootstrapDeviceCall (function) - Custom bootstrap handler
Returns:
Promise
